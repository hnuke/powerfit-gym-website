# Project Title

Powerfit Gym

## Description

This project is a responsive website built with HTML, CSS, and JavaScript. It showcases the fictious gym, their services, contact and etc.

## Demonstration

*   **Live Demo:** [link demo](https://powerfit-gym-website-kappa.vercel.app/index.html)
*   **Screenshot/GIF:** ![project screenshot](design/PowerFit Gymwebsite features.png)

## Technologies Used

*   **HTML5**
*   **CSS3**
*   **JavaScript (ES6+)**

## Installation

1.  Clone the repository to your local machine:
    ```bash
    git clone https://github.com/hnuke/powerfit-gym-website
    ```
2.  Navigate to the project directory:
    ```bash
    cd powerfit-gym-website
    ```
3.  Open the `index.html` file.

## Usage

The website is a simple static site. You can navigate through different sections using the navigation bar and view company details.
